# Project Overview

## Problem

Traffic sign detection is a practical object detection task with real-world constraints such as small objects, class imbalance, and varying lighting conditions.

## Approach

This project explores several YOLOv8-based variants and compares them across baseline and custom-module settings.

## Main experimental direction

- Compare a standard YOLOv8 baseline against custom lightweight variants
- Study feature-fusion changes rather than only scaling model size
- Examine robustness under day and night lighting conditions

## Key repository goals

- Make experiments reproducible
- Keep custom modules organized
- Keep training, evaluation, and prediction entrypoints simple
- Present the work clearly on GitHub
